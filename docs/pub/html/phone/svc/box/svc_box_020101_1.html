이용안내
  • IBK기업은행에서 운영하는 플랫폼(IBK BOX)을 기업뱅킹 로그인만으로 편리하게 이용할 수 있습니다.
  • 플랫폼에 가입하지 않은 경우 해당 플랫폼 회원가입을 해주세요.
  • 플랫폼 이용자 등록을 위해서 개인 휴대폰 본인인증과 플랫폼 비밀번호 등록을 해주세요.
  • 이후 다른 플랫폼을 등록할 때는 휴대폰 본인인증과 플랫폼 비밀번호를 확인해 주세요.
  • 등록한 플랫폼으로 이동할 경우 별도 로그인 없이 플랫폼 비밀번호 확인만으로 이용할 수 있습니다.
  • 인증서 별로 한 명만 등록할 수 있습니다.
  • 등록한 본인이 아닌 경우 이용할 수 없습니다.
  • 회사 내에서 해당 플랫폼 이용이 허가된 경우만 이용해 주세요.
이용방법
  1. Step 01 회원가입(플랫폼 또는 기업뱅킹)
  2. Step 02 플랫폼 이용자 등록 (플랫폼 비밀번호 숫자 6자리 등록)
  3. Step 03 플랫폼으로 이동 시 플랫폼 비밀번호 입력
  4. Step 04 로그인 없이 플랫폼 이용 가능
