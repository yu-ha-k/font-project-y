1

금융거래용

세금계산서 업무에는 사용할 수 없어요.

금융인증서
  • 수수료 4,400원 (부가세 포함, 1년)
  • 은행, 카드, 보험, 증권 등 업무에 사용
  • 유효기간은 1년, 2년, 3년 중에 선택

공동인증서
  • 수수료 4,400원 (부가세 포함, 1년)

은행별 금융인증서
  • 수수료 무료 (유효기간 3년)
  • IBK기업은행에서만 사용할 수 있으며 이체 권한이 없는 ‘조회용 이용자’만 발급 가능
IBK인증서
  • 수수료 무료 (한시적 무료, 유효기간 1년)
  • IBK기업은행 전용 인증서(타행 사용 불가)
  • 기업뱅킹 거래, (개인사업자)조달청 사용 가능
