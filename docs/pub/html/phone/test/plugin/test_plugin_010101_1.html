임대차계약서 촬영

사업장 또는 집 임대차(전월세) 계약을 체결한 경우 해당 임대차계약서 2부(사업장, 집)를 모두 촬영해 제출해 주세요.

본인이 아닌 가족이 임대차(전월세) 계약을 체결한 경우 가족 여부를 확인할 수 있는 ‘가족관계증명서’를 추가로 촬영해 제출해 주세요.

촬영할 서류가 여러 장인 경우 [촬영하기]를 눌러 추가촬영을 해주세요.

촬영할 서류가 10매가 넘는 경우 10매 촬영서류를 먼저 제출한 뒤 다시 [촬영하기]를 눌러 추가촬영을 해 주세요.
