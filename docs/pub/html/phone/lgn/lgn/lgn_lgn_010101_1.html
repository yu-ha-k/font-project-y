인증서를 어디에서 가져오시겠어요?

IBK기업은행 기업인터넷뱅킹에서 가지고 오려면 [PC에서 가져오기]를, 다른 스마트폰이나 앱에서 가지고 오려면 [스마트폰에서 가져오기]를 이용해 주세요

  • PC에서
    가져오기
  • 스마트폰에서
    가져오기
  • 기존의 i-ONE뱅크 기업 앱에서 공동인증서를 이용하시는 고객님은 새로운 앱으로 인증서를 가져오셔야 합니다. [스마트폰에서 가져오기]로 인증서가 설치된 기존 앱에 인증번호만 입력하면 손쉽게 복사해올 수 있습니다.
