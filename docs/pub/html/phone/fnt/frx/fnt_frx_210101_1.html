복잡한 외화송금을
QR코드 스캔 하나로 간편하게
QR코드 반복송금이란?
  • 영업점에서 받은 송금명세서(Remittance Details) 하단의 QR코드 스캔 시 해당 송금정보가 자동 입력돼 간편하게 반복송금할 수 있는 서비스예요.
이용 방법
  1. 1
    기존 송금명세서 준비
    송금명세서(Remittance Details)가 없는 경우 영업점에 요청 가능
  2. 2
    QR코드 스캔
    스마트폰 카메라 또는 i-ONE Bank 앱 내 'QR코드 반복송금’으로 스캔
  3. 3
    송금정보 자동 입력
    기존 송금정보 확인 및 송금금액 입력
  4. 4
    송금완료
    외화송금내역조회를 통해 거래내역 확인
서비스 이용시간 안내
송금종류, 전액 외화거래, 포지션 거래로 이루어진 서비스 이용시간 안내 표
